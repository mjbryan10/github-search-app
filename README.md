# GitHub Searcher

Utilising the Github API this app allows to search github for your favourite people.

For more information about the GitHub API, check out their documentation [here](https://developer.github.com/v3/).

## Technologies

* React
* Javascript
* Sass (SCSS)
* AJAX

## Live Demo
